# InstantGitHub

InstantGitHub allows you to instantly search GitHub users and repositories with a fast and clean interface.

## Features

* Instant search for GitHub users and repositories
* Clean and fast UI built with Vite and TypeScript
* Easy-to-understand component structure
* Extendable service and context layers

## Clone the Project

```bash
git clone https://github.com/qzxturkiye/InstantGitHub.git
cd InstantGitHub
```

## Environment Setup

Create a `.env` file in the project root and add your API key:

```
GEMINI_API_KEY=your_gemini_api_key_here
```

This key enables AI-powered features inside the project.

## Installation

Install all required dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```

## Project Structure

* `components/` – UI components
* `contexts/` – State management with React Context
* `services/` – API and helper functions
* `App.tsx` – Main application entry
* `vite.config.ts` – Vite configuration
